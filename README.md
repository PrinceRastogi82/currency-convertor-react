# Currency Converter

A simple and responsive Currency Converter built using React and Tailwind CSS.  
It fetches real-time currency exchange rates from an API and allows users to convert currencies instantly.

---

## Features

- Convert currencies in real time
- Swap currencies
- Reusable InputBox component
- Responsive UI
- API integration using custom hooks
- Built with React Hooks

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- Currency Exchange API
